# entracopes

[entrascopes.com](https://entrascopes.com) by [Fabian Bader](https://cloudbrothers.info/en/) and [Dirk-jan Mollema](https://dirkjanm.io/) 
was used in the development of this application. In particular, I used the raw json sources backing the site for a number of purposes:

You can download them with the following command
```bash
wget https://entrascopes.com/firstpartyscopes.json \
    https://entrascopes.com/resources.json \
    https://entrascopes.com/bypasses.json \
    https://entrascopes.com/Classification_AppRoles.json
```
